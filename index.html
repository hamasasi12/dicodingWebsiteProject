HTML

Hyper Text Markup Language adalah bahasa markup yang digunakan untuk membuat halaman web. Dengan HTML, Anda dapat menentukan struktur dan konten dari sebuah halaman web, termasuk teks, gambar, link, dan elemen-elemen lainnya. HTML menggunakan tag-tag untuk mengatur bagaimana konten tersebut ditampilkan dan diatur di dalam browser web.

CSS

Cascading Style Sheet adalah bahasa yang digunakan untuk mendesain tampilan halaman web yang dibuat dengan HTML. Dengan CSS, Anda dapat mengontrol format, tata letak, dan tampilan visual dari elemen-elemen HTML, seperti warna, font, ukuran, dan posisi. CSS memungkinkan pemisahan antara struktur (HTML) dan tampilan (CSS) dari sebuah halaman web, sehingga memudahkan untuk membuat tampilan yang konsisten dan menarik secara visual.

JS

Javascript adalah bahasa pemrograman yang digunakan untuk membuat halaman web interaktif. Dengan JavaScript, Anda dapat menambahkan fungsi, efek animasi, validasi formulir, dan interaksi dinamis lainnya ke dalam halaman web. Ini memungkinkan pengguna untuk berinteraksi dengan konten secara real-time tanpa harus memuat ulang halaman. JavaScript sering digunakan bersama dengan HTML dan CSS untuk membuat pengalaman web yang lebih dinamis dan responsif.

PHP

Hypertext PreProcessor adalah bahasa pemrograman server-side yang digunakan untuk membuat situs web dinamis. Dengan PHP, Anda dapat melakukan berbagai tugas seperti memproses formulir, mengakses database, menghasilkan konten dinamis, dan berinteraksi dengan pengguna. PHP berjalan di sisi server, artinya kode PHP dieksekusi di server web sebelum hasilnya dikirimkan ke browser pengguna, sehingga memungkinkan pembuatan situs web yang responsif dan berinteraksi dengan data.
